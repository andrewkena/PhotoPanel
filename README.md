# PhotoPanel 1.2 — плагин Mission Planner

Плагин для Mission Planner (проверено на MP 1.3.83), который считает количество
сделанных фотоснимков во время полёта и показывает телеметрию, связанную со
съёмкой: время и координаты последнего снимка, высоту в момент съёмки,
скорость, спутники/точность GPS, средние интервалы между снимками и данные о
следующей путевой точке.

Автор: andrewkena
Версия: 1.2

## Два варианта плагина

В проекте два файла — это **два разных плагина с одной и той же логикой**,
различается только способ отображения. **В папку `plugins` кладите только
один из них** — иначе оба плагина будут одновременно подписываться на одни и
те же MAVLink-сообщения для одного аппарата (конфликт подписок).

## Что показывает панель

| Поле | Описание |
|---|---|
| Снимков сделано | Счётчик фотографий с начала сессии (крупным шрифтом) |
| Время | Время последнего снимка (`ЧЧ:ММ:СС`) |
| Последнее фото | Координаты последнего снимка (широта, долгота) |
| Высота последнего снимка | Относительная высота (`alt_rel`) в момент съёмки, а не текущая высота аппарата |
| Скорость | Текущая воздушная скорость (м/с) |
| Спутники | Количество спутников, HDOP, VDOP |
| Среднее время между фотографиями | Средний интервал между снимками, сек |
| Среднее расстояние между фотографиями | Среднее расстояние между точками съёмки, м (гаверсинус) |
| Следующий WP | Номер следующей путевой точки маршрута |
| Расстояние до следующего WP | Расстояние до неё, м |

Кнопка **"Сброс"** обнуляет счётчик и всю накопленную статистику (но не
отключает подписку на телеметрию — счёт начинается заново).

## Источники данных (MAVLink)

- **CAMERA_FEEDBACK** — при получении каждого сообщения увеличивается счётчик
  снимков, сохраняются координаты (`lat`, `lng`), относительная высота
  (`alt_rel`) и время получения; на основе разницы с предыдущим снимком
  считается интервал по времени и расстояние (формула гаверсинуса).
- **GPS_RAW_INT** — поле `epv` используется как VDOP (значение `65535`
  трактуется как "нет данных").
- Скорость, спутники, HDOP, номер и расстояние до WP берутся из текущего
  состояния борта (`Host.cs`), а не из CAMERA_FEEDBACK.

Подписки идут только на текущий `sysid`/`compid` (`Host.comPort.sysidcurrent`
/ `compidcurrent`), зафиксированный на момент загрузки плагина.

## Установка

1. Скопировать **один** из файлов (`Photopanel.cs` или `PhotoPanelWindow.cs`)
   в `C:\Program Files (x86)\Mission Planner\plugins\`.
2. Перезапустить Mission Planner — плагин компилируется самим Mission Planner
   при старте, отдельная сборка (`dotnet build`) не требуется.
3. Подключиться к аппарату (реальному или SITL) — вкладка/окно появится
   автоматически.

## Обновление проекта в файле

- `bin/`, `obj/`, `.claude/` не входят в репозиторий (см. `.gitignore`) —
  сборка через `dotnet` не нужна, эти файлы для Mission Planner не имеют
  значения.
- Оба плагина используют один и тот же алгоритм подсчёта расстояния и статистику
  — при изменении логики в одном файле стоит проверить, нужно ли то же
  изменение во втором.
